# References

Use this page for the complete list of sources cited across the IMRAD manuscript.

## IEEE Reference Instructions

- Include only sources that are actually cited in the study.
- Use IEEE citation format consistently for all entries.
- Number the references in the order they first appear in the manuscript.
- Update this page whenever a new in-text citation is added to Chapters 1 to 5 or the appendices.

## In-Text Citation Guide

Use bracketed reference numbers in the manuscript, for example:

- single source: [1]
- multiple sources: [2], [3]
- consecutive sources: [4]-[6]

Do not use author-date citations if the manuscript is following IEEE format.

## In-Text Citation Check

Before final submission, verify the following:

- every in-text citation has a matching entry on this page
- every entry on this page is cited at least once in the manuscript
- reference numbers appear in first-citation order
- author names, publication years, titles, page numbers, and links or DOIs are accurate
- punctuation, abbreviations, and ordering follow IEEE style consistently

## Writing Reminder

Use official IEEE formatting for the final version, especially for journal titles, conference names, edition labels, page ranges, and online sources.
